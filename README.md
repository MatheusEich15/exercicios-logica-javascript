## 📌 Diferenciais Técnicos
Para estes exercícios, foquei em aplicar conceitos fundamentais da linguagem:
* **Manipulação de Dados:** Uso de tipos primitivos, objetos e arrays.
* **Estruturas de Controle:** Implementação de condicionais (`if/else`, `switch`) e laços de repetição (`for`, `while`).
* **Clean Code:** Variáveis com nomes semânticos para facilitar a leitura do raciocínio lógico.
