## 🔗 Sobre o Repositório
Este projeto contém a resolução de desafios lógicos propostos em aula, focando na construção de um raciocínio algorítmico sólido e eficiente. Cada lista aborda diferentes níveis de complexidade e estruturas da linguagem.

## 📌 Diferenciais Técnicos
Para estes exercícios, foquei em aplicar conceitos fundamentais da linguagem:
* **Manipulação de Dados:** Uso de tipos primitivos, objetos e arrays.
* **Estruturas de Controle:** Implementação de condicionais (`if/else`, `switch`) e laços de repetição (`for`, `while`).
* **Clean Code:** Organização em funções com nomes semânticos e interfaces de menu interativas.

## 🚀 Como executar localmente
1. Certifique-se de ter o **Node.js** instalado.
2. Navegue até a pasta e execute qualquer uma das listas utilizando o Node:
   ```bash
   node "Lista 1.js"
   # ou
   node "Lista 2.js"
   ```
